# Simple Calculator

A basic Windows Forms calculator application built with C# and .NET 8.0.

## Features

- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

## Requirements

- .NET 8.0 SDK or later
- Windows OS

## How to Run

1. Make sure you have the .NET SDK installed
2. Open a terminal in the project directory
3. Run the application:
   ```bash
   dotnet run
   ```

## Usage

1. Enter the first number in the first text box
2. Enter the second number in the second text box
3. Click one of the operation buttons (+, −, ×, ÷)
4. The result will be displayed below
